About Our Port Scanner Project

This project is developed as part of a college assignment, aiming to create a user-friendly web interface for network port scanning. It integrates the powerful Nmap tool with a Python backend (Flask) to provide quick and comprehensive port scanning capabilities.

Our goal is to demonstrate the practical application of web technologies combined with network security tools, offering insights into how services communicate over a network.

Technologies Used: HTML, CSS, JavaScript, Python (Flask), Nmap.
